Cocos2d-JS アドベンチャーパートエンジン
====

このプロジェクトは Cocos2d-JS で動作する簡易アドベンチャーエンジンです。同一フォルダ以下の resources/dataJson.json のシナリオを読み込んで動作します。
世界中のゲームのアドベンチャーパートで使われるように鋭意アップデート中です！

## 説明

Cocos2d-JS で動作するアドベンチャパートです。
